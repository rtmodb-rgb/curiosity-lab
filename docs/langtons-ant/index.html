Langton’s ant: order out of chaos

interactive — watch it build the highway verified — 30 + 19 checks sourced — proven · cited · open · empirical, flagged

One ant on a grid of square cells. Two rules. For about ten thousand steps it wanders in what looks like pure noise — then, abruptly, it locks into building a perfectly periodic diagonal “highway” and drives down it forever. It can be proved that it never gets permanently stuck; whether the highway must always appear is, astonishingly, still open. Below you can watch the whole story unfold, step by exact step.

1The whole rule fits in two lines

The grid starts all white. A single ant sits on one cell, facing up. At every step it looks only at the colour of the cell it is standing on, turns, flips that cell, and walks one square forward:

on a WHITE cell → turn right 90°, paint it black, step forward on a BLACK cell → turn left 90°, paint it white, step forward

That is the entire program. No randomness, no memory, no look-ahead — the ant cannot “see” anything but the square under its feet. Everything on this page is just this rule, iterated.

turn RIGHT ↻ — flip the cell to black, then step forward
turn LEFT ↺ — flip the cell to white, then step forward

One convention, fixed for the whole page

Sources state the rule with different left/right and up/down choices, which only mirrors the picture. We fix one: white → right, black → left; x runs east, y runs down (screen coordinates); the ant starts at the origin facing up. Because of this choice the highway runs off toward the lower‑left. The direction is convention‑dependent — swap left/right or flip the y‑axis and the signs change — but the numbers we headline (period 104, drift 2√2, +12 cells) are the same in every convention.

3The one big theorem: it can never get stuck

Here is the deepest thing that is proven. No matter how you colour a finite number of starting cells, the ant can never be trapped in a bounded region forever — it must visit infinitely many cells and wander off to infinity. It cannot fall into a small repeating loop in place. This is the cited unboundedness theorem of Bunimovich & Troubetzkoy (1992), proved by viewing the ant as a particle bouncing through a “Lorentz lattice gas” of mirrors. Troubetzkoy (1997) later strengthened it: the set of cells visited infinitely often has no corners.

What this theorem does not say

Unboundedness says the ant always escapes — it says nothing about how. It does not prove the highway must form, nor that the long‑run motion is the period‑104 drift. Escaping to infinity in some other, non‑highway way is not ruled out by this theorem. (That gap is exactly the open question in §4.)

A naming caution: MathWorld calls this the “Cohen–Kung theorem,” but that attribution is disputed — Wikipedia flags it as a misattribution (traced to a 1994 Scientific American column by Ian Stewart). The peer‑reviewed result of record is Bunimovich–Troubetzkoy (1992); we cite it that way.

4The open question: must the highway always appear?

Every all‑white run anyone has ever computed eventually builds the highway. Start instead from a few black cells, or a random blob, and — in practice — you still get a highway, sometimes after a much longer scramble. But “always” has never been proved. Paint some cells and try to find a counterexample (you won’t — but nobody can prove you can’t).

Open conjecture

Whether the ant builds a highway from every finite starting configuration is an open problem, stated as open by Gajardo, Moreira & Goles (2002). Keep it separate from the empirical fact that every run we’ve watched does highway. A tempting heuristic — “because the rule is reversible, it should highway” (§5) — is just that: a heuristic, not a proof.

5Reversibility: the rule runs backwards perfectly

The update is a bijection: from the ant’s position, heading and the colour it just left, you can always reconstruct the previous state. So you can run the film in reverse. Build a trail forward, then play it backwards and watch every black cell flip back to white until the grid is pristine and the ant is home, facing up — round‑trip error exactly zero. This is proven by code: the gate runs forward then inverse and checks the start is recovered exactly.

6Beyond two colours: turmites

Generalize the ant to n colours that cycle 0→1→…→n−1→0, with a rule‑string of n letters telling it which way to turn on each colour (R = right, L = left). Langton’s ant is simply the rule‑string RL. Tiny changes give wildly different worlds.

A beautiful cited theorem (Gale, Propp, Sutherland & Troubetzkoy, 1995): rule‑strings whose cyclic form is built only from LL and RR pairs — such as LLRR, RLLR, LRRL — return to bilateral symmetry infinitely often. Others, like RLR, grow chaotically with no highway known (open / empirical); LRRRRRLLR famously fills a tidy square (empirical).

7It can compute

The ant is not just a curiosity — it can do logic. cited Gajardo, Moreira & Goles (2002) showed the ant can simulate arbitrary Boolean circuits, and that deciding whether the ant ever visits a given cell is P‑hard for finite configurations — so the dynamics are, in a precise sense, as hard as any efficient computation. With infinite but finitely‑described initial configurations the ant is capable of universal computation, which makes long‑term prediction undecidable in general. One careful caveat: universality and undecidability rely on those infinite configurations; for genuinely finite starts the proven statement is P‑hardness, not Turing‑completeness.

8How we checked

Two gates back this page. verify.py (30 checks, standard‑library Python only) runs two independently written simulators — one a dictionary grid, one a set of black cells with its own movement code — and asserts they agree cell‑for‑cell for the first 12,000 steps, then pins the period, drift, cells‑per‑period, onset, reversibility and turmite facts. test_core.mjs (19 checks) runs the same JavaScript engine this page uses, cross‑checks it against Python, and — crucially — verifies that the engine inlined into this page is byte‑for‑byte identical to the tested file. The code is open: github.com/rtmodb-rgb/curiosity-lab.

Re‑computed here (two independent ways where it matters)

  • Two simulators reproduce the identical trajectory — position, heading and flipped cell — for 12,000 steps.
  • The highway has minimal period 104 (and P = 52 fails); net drift magnitude² = 8, i.e. 2√2 ≈ 2.828 cells per period; +12 black cells per period, checked across consecutive period boundaries.
  • Reversibility: forward 5,000 steps then 5,000 inverse steps returns to the exact origin — empty grid, position (0, 0), heading up.
  • Highway onset from all‑white lands at step ≈9,977 under a precisely defined detector (about 10⁴), inside a sane 9,000–11,000 band.
  • Turmites: the rule‑string RL reproduces Langton (period 104); the proven‑symmetric family LLRR / RLLR / LRRL returns to a verified bilateral symmetry.

Cited, not re‑proved

  • Unboundedness for any finite configuration — Bunimovich–Troubetzkoy (1992); strengthened by Troubetzkoy (1997).
  • Universality / P‑hardness / undecidability — Gajardo–Moreira–Goles (2002), with the finite‑vs‑infinite caveat above.
  • The bilateral‑symmetry theorem for LL/RR rule‑strings — Gale–Propp–Sutherland–Troubetzkoy (1995). We verify a concrete symmetry occurs; the “infinitely often” is theirs.
  • That the highway emerges from every finite start is open; “all‑white → highway” and “RLR is chaotic” are empirical.
