The house rules

Compute, don't assert

If the page states a number we computed, code on the page (or in the open repo) reproduces it. Literature results are cited, not claimed as our own.

Two methods or it didn't happen

Key results are checked by two independent routes that must agree to the digit.

Honest about the edges

Where the textbook story over-simplifies, the page says so plainly — sources and caveats included.

You can re-run it

The simulations are live and the reference code is open at github.com/rtmodb-rgb/curiosity-lab.

Rule 30: Is it random?

One line of cells and one tiny rule — new = left XOR (centre OR right) — make a pattern that has resisted explanation for 40 years and carries a $30,000 prize. We set a rule we can prove (Rule 90 is Pascal's triangle mod 2) right next to the open question, and let you run all 256 rules in the browser.

No. 03 Live

The Hat: one shape that never repeats

In 2023 four people found a single 13-sided tile that covers the whole plane but never periodically — the long-sought “einstein”. Grow it with its inflation rule, colour it by the four metatiles, and watch the tile frequencies converge to φ⁴. We check every headline number two ways; the page is honest about what's proven vs. cited.
